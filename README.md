# Afghan Returnees Reintegration Analysis

This repository presents a data-driven analysis of reintegration outcomes, resilience, and migration dynamics among Afghan returnees.

## Overview

This project combines qualitative and quantitative approaches, including:

- Exploratory Data Analysis (EDA)
- Factor Analysis
- Structural Equation Modeling (SEM)

## Data

Due to ethical and confidentiality considerations, raw data is not publicly shared. Simulated or processed data may be provided where appropriate.
